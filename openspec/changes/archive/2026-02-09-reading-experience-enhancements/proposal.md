## Why

Rosydawn 目前只提供浅色主题，缺少长文阅读辅助功能（进度提示、预计时间）和文章间导航，导致用户在长文阅读和连续浏览时体验不够流畅。CSS 变量体系已就位，扩展成本低，适合一次性补齐这四项阅读体验能力。

## What Changes

- **暗色模式**：基于现有 CSS 变量体系，新增 `prefers-color-scheme: dark` 自动适配，并提供手动切换开关，支持用户偏好持久化（localStorage）。
- **阅读进度条**：页面顶部固定一条细进度条，根据滚动位置实时显示阅读百分比，帮助用户感知长文进度。
- **预计阅读时间**：根据文章正文字数（中文按字符计数）在运行时计算预计阅读时间，显示在文章标题下方的元信息区域。
- **上/下篇导航**：文章底部添加前一篇 / 后一篇文章链接（按发布日期排序），降低跳出率，引导连续阅读。

## Capabilities

### New Capabilities

- `dark-mode`: 暗色主题支持，包括暗色 CSS 变量集、`prefers-color-scheme` 媒体查询适配、手动切换开关组件、偏好持久化
- `reading-progress`: 页面顶部阅读进度条，监听滚动事件实时更新宽度百分比
- `reading-time`: 基于文章字数的预计阅读时间计算与显示
- `prev-next-nav`: 文章详情页底部的上一篇/下一篇导航链接

### Modified Capabilities

_(无现有 spec 需要修改)_

## Impact

- **布局组件** (`src/layouts/Layout.astro`)：新增暗色变量集、进度条元素、主题切换开关
- **文章详情页** (`src/pages/blog/[...slug].astro`)：新增阅读时间显示、上/下篇导航、进度条脚本
- **样式**：所有使用 CSS 变量的组件自动适配暗色模式，可能需要微调个别组件在暗色下的表现（代码块、图片、边框等）
- **无新增依赖**：所有功能使用原生 CSS 和 vanilla JS 实现，保持零额外运行时依赖
- **无 API 变更**：纯前端增强，不影响构建产物结构和部署流程
