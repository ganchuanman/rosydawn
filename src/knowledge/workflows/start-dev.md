# 启动开发服务器 (start-dev)

启动本地开发服务器进行实时预览。

---

## 用途

在本地启动开发服务器：
- 热更新（HMR）
- 实时预览
- 调试模式

---

## 参数

无参数

---

## 使用示例

### REPL 模式

```
> 启动开发服务器
> 我想本地预览博客
```

### 命令行模式

```bash
# 启动开发服务器
rosydawn dev

# 完整命令
rosydawn dev:start
```

---

## 执行流程

1. 启动 Astro 开发服务器
2. 打开浏览器（默认 http://localhost:4321）
3. 启用热更新（文件修改自动刷新）

---

## 访问地址

- **本地**: http://localhost:4321
- **网络**: http://0.0.0.0:4321 （如果配置）

## 注意事项

- 默认端口 4321，可在 `astro.config.mjs` 修改
- 开发模式不进行完整构建，速度更快
- 修改 `src/` 下文件会自动刷新
