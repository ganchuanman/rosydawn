# 系统状态检查 (check-status)

检查 Rosydawn 系统的运行状态。

---

## 用途

检查系统健康状态：
- Git 状态
- 环境配置
- 文章统计
- 部署状态

---

## 参数

无参数

---

## 使用示例

### REPL 模式

```
> 检查系统状态
> 博客运行情况如何？
```

### 命令行模式

```bash
# 检查状态
rosydawn status

# 完整命令
rosydawn status:check
```

---

## 检查项目

### Git 状态
- 当前分支
- 是否有未提交的修改
- 远程仓库同步状态

### 环境配置
- Node.js 版本
- npm 版本
- 环境变量是否配置

### 文章统计
- 已发布文章数量
- 草稿数量
- 最近文章列表

### 部署状态
- 最后部署时间
- 构建版本
- Nginx 状态（如果在服务器上）

---
