# Assets 资源说明

本目录存放文章相关的静态资源文件。

## 文件列表

| 文件名 | 说明 | 建议尺寸 |
|--------|------|----------|
| `sdd-workflow.png` | SDD 工作流程图，展示从 Spec 编写到代码生成的完整流程 | 1200x600 |
| `spec-example.png` | Spec 编写示例截图，展示规范的 Spec 文档格式 | 800x1000 |

## 图片制作建议

- 使用 [Excalidraw](https://excalidraw.com/) 或 [Mermaid](https://mermaid.js.org/) 制作流程图
- 保持风格统一，建议使用浅色背景
- 导出时选择 2x 分辨率以保证清晰度
